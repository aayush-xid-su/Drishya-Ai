# **App Name**: DRISHYA AI

## Core Features:

- Real-Time Object Detection: Detect objects in real-time from video streams using computer vision models. This service may use tool to decide when to incorporate information into the output based on context
- Facial Landmark Detection: Identify facial landmarks with high accuracy, enabling face tracking and analysis.
- Hand Gesture Recognition: Recognize hand gestures to provide interactive control and analysis of user inputs.
- Pose Estimation: Estimate human poses in real-time, capturing body movements and positions.
- Live Stream Processing: Enable live processing of video streams from various sources (e.g., webcams, IP cameras).
- Results Overlay: Overlay detection results on the video stream with clear visual indicators.
